# Rani Market

This is an e-commerce application built with JavaScript. Vue for the frontend and NodeJS, ExpressJs for the backend. The database is MongoDB.

## Installation

### 1. Clone project

### 2. Install dependencies
```bash
cd client
npm install
```
```bash
cd server
npm install
```

### 3. Run client
```bash
cd client
npm run serve
```

### 4. Run server
```bash
cd server
npm start
```
